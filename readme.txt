Displays a email signup box to capture email address to use with bronto.

== Description ==
 
 
A few people have been after a plugin that allows Bronto clients to capture email addresses for their newsletter.

This plugin will display an email signup box where users can enter their email address and that email will be added to a bronto list or lists of your choice

Note that for now this form will display using the footer hook.

 = Planned Updates = 
  * Shortcode to place the form anywhere
  * Form builder so you can edit the fields you want to input
  * A better design method


== Installation ==

Wordpress:
1. Search for Bronto Newsletter Signup
2. Click Install
3. Press Activate

Uploading:
1. Download sbd-bronto-newsletter.zip
2. Upload `sbd-bronto-newsletter` to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
